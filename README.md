# Locus Plugin Registry

Plugin registration PRs modify source entries under `entries/v1/plugins/<bucket>/<plugin-id>.json`.

The public registry API is generated by CI under `public/v1`:

- `public/v1/manifest.json`
- `public/v1/shards/<bucket>.json`
- `public/v1/plugins/<bucket>/<plugin-id>.json`
- `public/v1/search/summaries.json`

Locus clients should read the generated base URL, for example:

`https://raw.githubusercontent.com/r1n7aro/locus-plugin-registry/main/public/v1`
